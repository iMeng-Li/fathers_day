###2、页面层级的流失~~

父亲节H5一共10个页面的滑动。
这里使用了百度统计来统计，每个页面的触发都有记录。可以看得出来，随着页面的深层页面的流失度就越大。

基本的流程是这样。

而在我这里的曲线也符合了这个，不过这个父亲节H5的曲线岂不是这样的，它是左右滑动的，用户滑动到最后，发现滑动到了最后一页，
会往回滑动的。
